# Scrabble board recognition (Распознавание доски Scrabble)

### Необходимые пакеты:
#### _Python 3.7_
#### `tensorflow 2.2.0`
#### `keras 2.3.1`
#### `pillow 7.1.2`
#### `imutils 0.5.3`
#### `scipy 1.1.0`
#### `pyqt5 5.14.2`
